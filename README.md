# Modern Clothing Store

A modern and responsive fashion e-commerce website built using HTML, CSS, and JavaScript.  
This project provides a clean shopping experience with product browsing, cart management, category filtering, admin dashboard functionality, and order management features.

---

## Features

- Responsive modern user interface
- Product category filtering
- Shopping cart functionality
- Product detail page
- Admin dashboard
- Add/Delete product management
- Orders section
- Toast notifications
- Mobile-friendly navigation
- Interactive user experience using JavaScript

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
